# Ai_impact_on_humans_mental_growth-Prediction


## Overview

This project investigates the potential impact of Artificial Intelligence (AI) usage on human cognitive and psychological development using Machine Learning techniques.

The objective is to predict whether AI usage is associated with:

- Positive Growth
- Stagnant / Moderate Growth
- Cognitive Decline Risk

The model analyzes cognitive, behavioral, and demographic factors to classify the expected outcome.

---

## Problem Statement

As Artificial Intelligence becomes increasingly integrated into everyday life, understanding its influence on human mental growth has become an important area of research.

This project aims to develop a Machine Learning model that predicts the impact of AI usage on human cognitive development by analyzing factors such as creativity, critical thinking, emotional intelligence, learning speed, AI dependency, and mental wellbeing.

## Final Model

Logistic Regression was selected as the final model due to:

- Highest testing accuracy
- Highest cross-validation accuracy
- Minimal overfitting
- Strong generalization performance
